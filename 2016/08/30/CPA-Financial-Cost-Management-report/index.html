CPA财务成本管理 - 财务报表分析和财务预测

卡西莫多cc
2016-08-30
Laboratory
CPA, 财管

1.财务报表分析的目的

将财务报表数据转换为有用的信息

2. 财务报表分析的维度

  • 1.战略分析
    确定主要的利润动因及经营风险并定性评估公司盈利能力，包括宏观分析，行业分析和公司竞争策略分析等
  • 2.会计分析
    评估公司会计的灵活性、恰当性、修正会计数据等
  • 3.财务分析
    比率分析和现金流量分析等
  • 4.前景分析
    财务报表预测和公司估值等

3.财务报表分析的方法

  • 1.比较分析法
    趋势分析：与本企业的历史比较
    横向分析：与同类企业（行业平均数或对标企业）比较
    预算差异分析：实际执行结果与计划预算指标比较
    会计要素的总量比较分析：主要采用时间序列分析，有时也用于同业比较
    结构百分比比较分析：将财务报表转换成结构百分比报表，用于发现占比不合理的项目，揭示进一步分析的方向
    财务比率比较分析：财务比率是相对数，排除了规模的影响，计算相对简单，具有较好的可比性，但对其加以说明和解释却比较复杂和困难
  • 2.因素分析法
    第一步：确定需要分析的财务指标，比较其实际数额和标准数额之间的差异
    第二步：确定财务指标的驱动因素，建立财务指标与各驱动因素之间的函数关系模型
    第三步：确定驱动因素的替代顺序，并逐一替代
    第四步：按顺序计算各驱动因素脱离标准的差异对财务指标的影响

5.财务比率分析

5.1 偿债能力分析

分析方法：1. 存量比较分析法 2.流量比较分析法

  • 1.短期偿债能力分析
    营运资本 = 流动资产 - 流动负债 = (总资产 - 非流动资产) - (总资产 - 股东权益 - 非流动负债) = (股东权益 + 非流动负债) - 非流动资产 = 长期资产 - 长期资产 => 本质上是长期资本结余
    营运资本配置比率 = 营运资本 / 流动资产
    流动比率 = 流动资产 / 流动负债 = 1 / (1 - 营业资本配置比率) = 1 / (1 - 营运资本 / 流动资产)
    速动比率 = 速动资产 / 流动负债
    现金比率 = 货币资金 / 流动负债
    现金流量比率 = 经营活动现金流量净额 / 流动负债
    增强短期偿债能力的其他因素：1.可动用的银行授信额度 2.可快速变现的非流动资产 3.偿债能力的声誉
    降低短期偿债能力的其他因素：1.与担保有关的或有负债事项
  • 2.长期偿债能力分析
    资产负债率 = 总负债 / 总资产
    产权比率 = 总负债 / 股东权益
    权益乘数 = 总资产 / 股东权益 = 产权比率 + 1 = 1 / (1 - 资产负债率)
    长期资本负债率 = 非流动负债 / (非流动负债 + 股东权益) = 非流动负债 / 长期资本
    利息保障倍数 = 息税前利润 / 利息费用 = (净利润 + 利息费用 + 所得税费用) / 利息费用
    现金流量利息保障倍数 = 经营活动现金流量净额 / 利息费用
    现金流量与负债比率 = 经营活动现金流量净额 / 总负债
    影响长期偿债能力的其他因素：1.在分析企业长期偿债能力时，应根据有关资料判断担保责任可能带来的影响 2.一旦判决败诉，可能会影响企业的偿债能力
5.2 营运能力分析
  • 应收账款周转率 = 营业收入 / 应收账款
  • 存货周转率 = 营业收入 / 存货
  • 流动资产周转率 = 营业收入 / 流动资产
  • 营运资本周转率 = 营业收入 / 营运资本
  • 非流动资产周转率 = 营业收入 / 非流动资产
  • 总资产周转率 = 营业收入 / 总资产
5.3 盈利能力分析
  • 营业净利率 = 净利润 / 营业收入
  • 总资产净利率 = 净利润 / 总资产 = 营业净利率 * 总资产周转率
    总资产净利率是公司盈利能力的关键，提高权益净利率的基本动力是总资产净利率
  • 权益净利率 = 净利润 / 股东权益 = 总资产净利率 * 权益乘数 = 营业净利率 * 总资产周转率 * 权益乘数
    概括了公司的全部经营业绩和财务业绩，可以衡量企业的总体盈利能力
5.4 市价比率分析
  • 市盈率 = 每股市价 / 每股收益
    每股收益 = (净利润 - 优先股股息) / 流通在外的普通股加权平均股数
  • 市净率 = 每股市价 / 每股净资产
    每股净资产 = 普通股股东权益 / 流通在外的普通股加权平均股数
  • 市销率 = 每股市价 / 每股营业收入
    每股营业收入 = 营业收入 / 流通在外的普通股加权平均股数
