1.穷人与富人的本质区别

富人得到资产，穷人得到负债

2.资产与负债

  • 资产是能把钱放进你口袋的东西
    股票、基金、房产、知识产权、版税等等.

  • 负债是把钱从你口袋里取走的东西
    信用卡透支、折价的车子等

3.穷人、中产与富人的现金流

  • 穷人
    拼命工作，努力赚钱，日常生活开销占据了收入的大部分，资产栏空空如也.

  • 中产
    收入较高，但是基本用于购买汽车，衣服，奢侈品，房租，偿还银行贷款. 工作 => 赚钱 => 支出 => 再工作 => 再赚钱 => 再支出 => …

  • 富人
    长期积累的资产所产生的的收入，轻而易举地覆盖掉日常的花销. 被动收入 > 必要开销，实现真正的财务自由.
