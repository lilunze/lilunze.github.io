1.股票概念

股票是一种有价凭证，公司发股票就是拿企业部分的所有权换得资金，用于后续的经营发展；当你花钱买入股票，拥有了公司一部分股权，也就成为了公司的股东。

2.开户

想要进行股票交易，需到开通证券公司账户，不同的交易板块的准入门槛也不一样，主板门槛相对较低，只需要身份证、储蓄卡就可以开通:平安证券 广发证券 等

4.交易制度

国内采取T+1交易制度，即当日买入的股票，需要等到下一个交易日才能卖出
交易时间: 工作日 9:30 - 11:30 13:00 - 15:00
集合竞价：
涨跌停制度: 每一个交易日中，单支普通股票的交易价格相对上一交易日收盘价格的涨跌幅度不得超过10%. 例外：1.ST开头的股票涨跌幅限制为5% 2.新股上市第一天无限制 3.停牌公司复牌第一天无限制

5.常见概念

  • 优先股和普通股
    优先股有固定的股息率，不参与红利分配，公司破产清算时优先清偿，优先分配股利，一般对公司无决策表决权，属于混合筹资，兼有债务筹资和股权筹资性质，一般由公司定向增发.
    普通股就是我们常见的上市交易的股票.

  • 股息、分红、股利
    股息是指股票的利息，一般针对优先股将公司的净利润按照固定的股息率分配给股东
    分红是上市公司向普通股股东分发现金股利和股票权利
    股利是股息与分红的简称

  • 市值与流通市值
    市值是该公司当前的股票价格乘以该公司所有普通股的股数
    流通市值是只特定的时间段内，可交易的流通股股数乘以当时的股价得出的流通股票总价

  • PE、PB、PS、ROE及其分点位
    PE(市盈率) = 市值 / 净利润
    PE(TTM)滚动市盈率 = 市值 / 过去12个月净利润
    PE静态市盈率 = 市值 / 上一年净利润
    PE动态市盈率 = 市值 / 推算净利润(根据最近几个季度的净利润推算出的净利润)
    PB市净率 = 股价 / 每股净资产
    PS市销率 = 总市值 / 销售收入
    ROE净资产收益率 = 净利润 + 净资产
    分点位表示当前指标在过去10年中所处的一个位置，例如：PE 80%分点位表示过去10年PE最大值和最小值区间的80%位置处的值

  • 股息率
    股息率 = 每股分红 / 每股股价
    股息率越高意味着分红越多，背后的投资逻辑就是公司的经营状况良好，盈利能力较好

  • 分红率
    分红率 = 分红金额 / 净利润
    分红率越高意味着公司盈利部分分给股东的金额越高，对股东有利

  • 市值加权、等权、平均值、中位数
    市值加权：所有样本公司的市值之和 / 所有样本公司净利润之和
    等权：n / n∑(1 / PE.i)
    平均值：以PE为例，算出所有样本公司的市盈率，使用四分位距去除极端值然后求出平均值
    中位数：以PE为例，算出所有样本公司的市盈率排序，取中间数；如果为偶数，取中间两数求平均值

7.投资建议

  • 1.定投高分红(高股息率，高分红率)股票
  • 2.参考A/H股比价，溢价率越高适合投资港股，溢价率低适合投资A股
