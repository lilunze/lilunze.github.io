1.复利的增长曲线

假设你在20岁那年存下1.4万用于投资，年化收益率为20%，那么，等到60岁退休时，这笔钱最终变成多少？
根据复利公式：F = A * (1 + i)^n
F = 1.4 * (1 + 0.2)^40 = 2057.68 万

2.复利最需要的是时间

根据上面的公式可以得出
F = 1.4 * (1 + 0.2)^4 = 2.903
也就是说要想真切看到复利的效果，也就是资产翻倍的时间最少是4年，72法则也是同样的道理，它能快速计算复利投资资产翻倍的时间
D = 72 / i
在投资年化收益20%的前提下投资10万元，经过D = 72 / 20 = 3.6年后，资产将变成20万.

4.结余率

结余率 = （收入 - 支出）/ 收入
决定我们财富增长的除了时间就是投入，我们只有尽可能的增加收入，减少不必要的开支，这样我们每个月投入的钱也就越多，财富增长也就越快，所以结余率越高越有利于财富增长.
